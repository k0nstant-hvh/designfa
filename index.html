S.A.N.J.A.R.

Secure Automated Network Joint Anti-Rogue

"Ваша цифровая граница под надежной защитой."

1) Описание проекта

S.A.N.J.A.R. — это интерактивный симулятор «ручного фаерволла», вдохновленный механикой Papers, Please. Вместо паспортов вы проверяете сетевые пакеты.

Проблема: Большинство людей не понимают, как работают угрозы в сети. Мы решаем это через геймификацию, обучая пользователей основам кибербезопасности.

Уникальность: В отличие от скучных лекций, наш проект дает почувствовать давление реальной атаки в реальном времени.

2) Функциональные требования

  • Анализ пакетов: Просмотр IP-адресов, портов и полезной нагрузки (payload).
  • Система фильтрации: Команды ACCEPT для чистых данных и DROP для угроз.
  • Система счета: Начисление баллов за правильную фильтрацию и штрафы за пропуск вирусов.
  • Динамическая сложность: Постепенное увеличение скорости трафика и сложности атак.

3) Технические требования

  • Язык программирования: C++ (MinGW) — для высокой производительности ядра.
  • Графика и Дизайн: Autodesk Maya — создание 3D-интерфейса терминала.
  • Хостинг: GitHub Pages (сайт) и GitHub Repository (исходный код).
  • Репозиторий: github.com/k0nstant/sanjar-project
