S.A.N.J.A.R.

Secure Automated Network Joint Anti-Rogue

1) Project Description

Project Overview

S.A.N.J.A.R. is a simulation of a manual network firewall. The user acts as an inspector who monitors and filters incoming data packets to maintain server security.

Problem Statement

Many students lack a practical understanding of how network security works. This project addresses the gap between complex theory and real-world application through an interactive environment.

Importance

Understanding digital threats is essential in the modern era. This tool helps users recognize suspicious data patterns before they cause actual damage.

Unique Value

The project focuses on high-pressure decision-making and technical logic, providing a realistic simulation of network defense tasks.

2) Functional Requirements

  • Real-time display of incoming data packet headers.
  • Manual "Accept" and "Drop" commands for filtering traffic.
  • Score tracking system based on successful filtering and security breaches.
  • Progression system with increasing difficulty and traffic volume.

3) Technical Requirements

  • Programming Language: C++ (MinGW compiler).
  • Version Control: Managed via GitHub.
  • Hosting: Website deployed through GitHub Pages.
