S.A.N.J.A.R.

Secure Automated Network Joint Anti-Rogue

1) Project Description

S.A.N.J.A.R. is an interactive "manual packet inspection" simulator inspired by the mechanics of Papers, Please. Instead of passports, you inspect network packets at the gateway of a high-security server.

The Problem: Most people lack practical understanding of how network threats work. Traditional education is often too theoretical and disengaging.

The Solution: This project teaches cybersecurity fundamentals through high-pressure gamification, training users to identify malicious patterns in real-time.

Uniqueness: It combines technical network logic with intense decision-making gameplay, creating a "human firewall" intuition.

2) Functional Requirements

  • Packet Inspection: Real-time analysis of IP addresses, ports, and data payloads.
  • Filtering System: Use ACCEPT to allow safe traffic and DROP to block malicious threats.
  • Scoring Mechanism: Gain points for accurate filtering; lose system health for allowing breaches.
  • Dynamic Threat Scaling: Traffic speed and attack complexity increase as the session progresses.

3) Technical Requirements

  • Core Engine: Built in C++ using the MinGW compiler for high-performance system simulation.
  • Design & Assets: 3D environment and terminal interface modeled in Autodesk Maya.
  • Platform: Native Windows Desktop Application.
  • Hosting: Website hosted via GitHub Pages.
  • Source Code: Managed on GitHub.
