S.A.N.J.A.R.

Secure Automated Network Joint Anti-Rogue

1) Project Description

What it is

S.A.N.J.A.R. is an interactive "manual packet inspection" simulator. Users take on the role of a system firewall, manually verifying data requests to protect a server infrastructure.

The Problem

Standard cybersecurity education is too theoretical. Students often fail to recognize real-time digital threats like DDoS or malicious code because they lack practical experience.

The Solution

By gamifying the inspection process, we provide a hands-on environment where users learn to identify attack patterns under pressure, building "digital intuition."

Uniqueness

Unlike simple games, this project focuses on realistic network logic and fast-paced decision making, making it a professional-grade educational tool.

2) Functional Requirements

  • Packet Analysis: Review IP addresses and data payloads in real-time.
  • Firewall Logic: Use "Accept" and "Drop" commands to filter traffic.
  • Scoring System: Performance-based metrics based on security accuracy.
  • Stress Simulation: Scalable difficulty based on incoming traffic volume.

3) Technical Requirements

  • Programming: C++ (MinGW) for high-performance processing.
  • UI Assets: Professional 3D modeling for the terminal interface.
  • Deployment: Managed and hosted via GitHub.
