S.A.N.J.A.R.

Secure Automated Network Joint Anti-Rogue

1) Project Description

Project Overview

S.A.N.J.A.R. is a simulation of a manual network firewall. The user acts as a 'firewall' who monitors and filters incoming data packets to maintain servers work.

Problem Statement

Many students lack a practical understanding of how network security works. This project will be trying to close the gap between hard theory and real-world application through a game-like environment.

Importance

Understanding digital threats is crucial in the nowadays era (especially with evolvement of AI). This tool will help users recognize suspicious data patterns before they cause actual damage to the users themselves.

Unique Value

The project focuses on high pressure decision-making and technical logic, providing a realistic simulation of network defense tasks.
